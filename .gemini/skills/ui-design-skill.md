# Testing Best Practices Skill

## Guidance
- **Test Strategy:** Prioritize testing core business logic and critical user flows.
- **Unit Tests:** Focus on testing individual functions and components in isolation.
- **Integration Tests:** Verify that different parts of the system work together as expected.
- **Reproduction:** Always create a reproduction test case when fixing bugs.
- **Maintainability:** Tests should be easy to read and maintain. Avoid over-testing implementation details.

## Workflow
- Write tests for all new features and bug fixes.
- Run existing tests frequently to catch regressions.
- Use mocks/stubs effectively for external dependencies.
